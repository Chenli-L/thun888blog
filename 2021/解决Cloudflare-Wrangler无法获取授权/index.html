解决Cloudflare Wrangler无法获取授权

字数统计: 104字   |   阅读时长: 1分

前言

昨天，因为要本地提交点东西到worker就装了Wrangler，没想到啊，它就这么卡着不动了（在waiting for api token那里我等了半个世纪）

方法

  • 在刚刚的授权页面点击API令牌

  • 点投放–copy

  • 输入wrangler config,粘贴刚刚copy的token

  • 回车后看到sucess就好了

后

写的时侯为了截图换了个账号，然后它能用了~~~~
