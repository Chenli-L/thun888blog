安装（GO）

环境

  • Go > 1.16

  • git

安装

1.fork ArtalkJS/ArtalkGo: 🗃 Artalk 评论系统后端程序 施工中… 仓库

2.cd到/ArtalkGo/目录下后，运行main.go以下载依赖（如果go版本<1.16就有报错）

Snipaste_2021-08-29_15-05-10

3.安装完后再运行一遍，不出意外应该是这样的

image-20210911094243644

配置

与php版本类似，把artalk-go.example.yml重命名为artalk-go.yml

打开来，设置下第8行的app_key（加密密钥）
